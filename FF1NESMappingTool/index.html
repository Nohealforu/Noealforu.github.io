MoveSpeed

Controls: arrow keys move, "x" key is what I have mapped to the interacting with things at this time (enter/exit airship at this point).
If you can't move, make sure the browser window is in focus (click on it). Customizable keybinds will be available in the future.
This controller image is also kind of touch enabled (not mouse)

Source code on the Github repository.
Art assets are from screenshots of FF1 NES version, property of Square Enix, they are not pulled from the original data. (i.e. spritesheets, 3 color pallete, etc.)
Code is based on some techniques from a Mozilla development article and does not use any portion of code from the FF1 NES Cartridge.
This free tool is for learning about the game systems and planning routes for Speedrunning, it is not a recreation of or substitute for the game.
If you're looking to play the original game, I highly recommend purchasing the Pixel Remaster version.
Thank you to Disch for the FFHackster tool for the ability to see how certain tile systems work so I could approximate portions of the game.
